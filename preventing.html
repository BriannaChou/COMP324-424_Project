Browser Hardening

"Hardening is a process whereby a computer is made more resistant to cyber intrusion from malicious attack and from accidental infection. [link]" This is a great definition of what hardening is. However the goals for this website are for hardening a browser to make it more private, which would allow it to be more secure.

Recommendations for Firefox Hardening

General Section:

Uncheck Restore previous session
- If you use a VPN and your connection goes down, then when Firefox loads up again, the website can potentially know your IP if you restore the page and are not using the VPN.

Home Section:

Set Homepage and new windows to Blank Page
- one less connection when you boot up the browser
Set New Tab to Blank Page
- These will prevent Firefox from spawning their own sites when you need a new page/tab.
Uncheck everything under Firefox Home Content (Shortcuts, Recommended by Pocket, Recent activity, Snippets)
- This will prevent Firefox from loading their own content
