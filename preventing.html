Browser Hardening

"Hardening is a process whereby a computer is made more resistant to cyber intrusion from malicious attack and from accidental infection. [link]" This is a great definition of what hardening is. However the goals for this website are for hardening a browser to make it more private, which would allow it to be more secure.

Recommendations for Firefox

General Section:

Uncheck Restore previous session
- If you use a VPN and your connection goes down, then when Firefox loads up again, the website can potentially know your IP if you restore the page and are not using the VPN.

Home Section:

Set Homepage and new windows to Blank Page
- one less connection when you boot up the browser
Set New Tab to Blank Page
- These will prevent Firefox from spawning their own sites when you need a new page/tab.
Uncheck everything under Firefox Home Content (Shortcuts, Recommended by Pocket, Recent activity, Snippets)
- This will prevent Firefox from loading their own content

Search Section

Change Default Search Engine to DuckDuckGo
- DuckDuckGo is one of the more privacy-centric browsers (searx is a good alternative as well)
Uncheck Provide Suggestions
- Prevents the queries from going to the Google API

Privacy & Security

Under Cookies and Site Data, check Delete cookies and site data when Firefox is closed
- next time the you visit the same website, the website will not be able to read the previous sessions cookies, since there none.
Under Logins and Passwords, uncheck the boxes (with the inside list first):
  • Show alerts about passwords for breached websites
  • Suggest and generate strong passwords
  • Autofill logins and passwords
  • Ask to save logins and passwords for websites
Under History, in the dropdown choose Firefox will Use custom settings for history
  • uncheck Remember search and form history, and Remember browsing and download history
  • Check Clear history when Firefox closes
  • DO NOT check the box next to Always use private browsing mode
  • - this will break Firefox containers
Under Address Bar, only have Bookmarks and Open tabs showing up
Under Permissions, click Settings... next to each Location, Camera, Microphone, and Notifications. In the popup, check the "Block new requests.." box on the bottom
Under Firefox Data Collection and Use, and uncheck everything
Under Security, uncheck everything
Under HTTPS-Only Mode best practice would be to use Enable HTTPS-Only Mode in all windows.
These are the settings when you enter about:config in the search bar:
    geo.enabled -> false
    Disables firefox from sharing your location
    browser.safebrowsing.malware.enabled -> false
    Disables Google's ability to monitor your web traffic for malware
    dom.battery.enabled -> false
    Blocks sending battery level information
